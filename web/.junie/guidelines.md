# Project Development Guidelines (instrlabs-apps/web)

## Directories
- `components/*`:reusable components
- `app/*`: pages
- `constants/*`: constant variables
- `hooks/*`: providers & hooks functions
- `services/*`: http services
- `utils/*`: utils functions
